# 2. 使い方

Arduino Nano を使ったプログラムを置いています。各プロジェクトにて詳細確認ください。

- LEDCube : LED Cube プロジェクト
- DFPlayerMini : DFPlayerMini と光センサーを使ったプロジェクト
- SSD1306 : SSD1306 を表示させるサンプルプログラム
- ST7735 : ST7735 を表示させるサンプルプログラム
